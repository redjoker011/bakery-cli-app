bakery-cli-app

Bakery Ordering CLI App built in Ruby and a powerful Ruby library for interactive cli app using tty-prompt

Output

Installation

  • Clone repository and go to project directory
  • Install dependencies via bundle install
  • Run app ruby bin/cli.rb

Specs

This app is covered with unit test using minitest. Run bundle exec rake test to check test cases under /test directory

Documentation

This app is well documented using yardoc. To view documentation locally follow yardoc installation guide and run documentation server via yard server and go to http://localhost:8808
